# Базовые упражнения по Python

Комплект устроен по папкам. В каждой папке есть два файла:

- `theory.md` — краткая теория по теме.
- `tasks.py` — задания для самостоятельной практики.

## Темы

1. `01_variables_types/` — переменные, типы данных, операции.
2. `02_input_output/` — ввод, вывод, форматирование строк.
3. `03_conditions/` — условия и логические выражения.
4. `04_loops/` — циклы `for` и `while`.
5. `05_strings/` — строки и методы строк.
6. `06_lists/` — списки, индексы, срезы, методы.
7. `07_dictionaries/` — словари и работа с парами ключ-значение.
8. `08_functions/` — функции, параметры, `return`.
9. `09_oop_basics/` — основы ООП: классы, объекты, атрибуты, методы.
10. `10_oop_models/` — классы-модели: состояние, валидация, методы предметной области.
11. `11_oop_interface_classes/` — классы для интерфейса: отделение логики от ввода-вывода.
12. `12_oop_principles/` — основные принципы ООП: инкапсуляция, наследование, полиморфизм, абстракция.
13. `13_oop_class_attributes/` — атрибуты класса и атрибуты объекта.
14. `14_oop_magic_methods/` — магические методы: `__str__`, `__repr__`, `__eq__`, `__len__`, `__lt__`.
15. `15_oop_properties/` — `property`, сеттеры и вычисляемые свойства.
16. `16_oop_class_static_methods/` — `classmethod`, `staticmethod` и альтернативные конструкторы.
17. `17_clothing_store_project/` — большой поэтапный проект: онлайн-магазин одежды с моделями, репозиториями, сервисами, корзиной, заказами и сохранением данных.
18. `18_tkinter_desktop_app/` — desktop-интерфейс на `tkinter` для проекта онлайн-магазина: окна, виджеты, события, каталог, корзина и оформление заказа.
19. `19_pytest_testing/` — тестирование проекта онлайн-магазина на `pytest`: unit-тесты, интеграционные тесты, happy path, negative path, edge cases, фикстуры и тестовая PostgreSQL-база.

## Как проводить занятие

1. Дайте студентам файл `theory.md` из нужной папки.
2. После разбора попросите открыть `tasks.py` из той же папки.
3. Каждое задание можно выполнять прямо в `tasks.py` под комментарием `TODO`.
4. Проверку лучше проводить запуском файла:

```bash
python 01_variables_types/tasks.py
```

Для самостоятельной работы можно просить студентов закомментировать уже выполненные задания или выносить решения в отдельные файлы.
